# create-node-base-kit

This package exists so `npm init node-base-kit` works on the npm registry. It runs the same scaffold as **node-base-kit**.

Install and use the main CLI instead unless you specifically want the init flow:

```bash
npx node-base-kit
```

Or, after this package is installed:

```bash
npm init node-base-kit
```

Documentation: [node-base-kit on npm](https://www.npmjs.com/package/node-base-kit) and [GitHub](https://github.com/AJKakarot/node-base-kit).
